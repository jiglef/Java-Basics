# Java Basics

Repositorio con proyectos y ejercicios básicos desarrollados en Java.

El objetivo de este repositorio es **afianzar los conocimientos adquiridos**, practicar los conceptos fundamentales del lenguaje y mejorar la lógica de programación mediante pequeños proyectos y ejercicios de dificultad progresiva.

## Contenido

Cada carpeta corresponde a un proyecto o conjunto de ejercicios sobre un tema concreto.

## Ejercicios - Bloque I

- Ejercicio  - Gestión de notas
Programa que solicita las notas de varios alumnos y calcula el número de aprobados, suspensos, la nota media, la nota más alta y la más baja.

- Ejercicio  - Menú Calculadora
Calculadora por consola con un menú interactivo que permite realizar operaciones básicas (suma, resta, multiplicación y división) utilizando métodos y estructuras de control.

- Ejercicio  - Estadísticas Array
Programa que almacena números en un array y obtiene estadísticas como suma, media, cantidad de números pares e impares, valor máximo y mínimo.

- Ejercicio  - Gestión de palabras
Programa que almacena palabras en un array, muestra su longitud, cuenta las palabras con más de cinco caracteres e identifica la palabra más larga.

- Ejercicio  - Gestión de alumnos
Programa orientado a objetos que permite registrar alumnos con su nombre, edad y nota final, mostrar sus datos, calcular el número total de aprobados e identificar al alumno con la nota más alta.

- Ejercicio - Gestión de peliculas
Aplicación de consola orientada a objetos que permite registrar películas, mostrar un catálogo, buscar películas por título, recomendar las mejor valoradas y calcular estadísticas como la duración media y la película con mayor puntuación.

- Ejercicio  - Gestión de animales
Programa que gestiona distintos tipos de animales mediante herencia, mostrando sus datos y su comportamiento específico a través de la sobrescritura de métodos.
