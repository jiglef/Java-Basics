# Java Basics

Repositorio con proyectos y ejercicios básicos desarrollados en Java.

El objetivo de este repositorio es **afianzar los conocimientos adquiridos**, practicar los conceptos fundamentales del lenguaje y mejorar la lógica de programación mediante pequeños proyectos y ejercicios de dificultad progresiva.

## Contenido

Cada carpeta corresponde a un proyecto o conjunto de ejercicios sobre un tema concreto.
